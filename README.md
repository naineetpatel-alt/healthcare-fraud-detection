
## ğŸ“Š API Endpoints

### Authentication
- `POST /api/v1/auth/register` - Register new user
- `POST /api/v1/auth/login` - Login and get tokens
- `POST /api/v1/auth/logout` - Logout and revoke token
- `GET /api/v1/auth/me` - Get current user info

### Fraud Detection
- `POST /api/v1/fraud/detect` - Analyze claims for fraud
- `GET /api/v1/fraud/health` - Check API health

### Dataset
- `GET /api/v1/dataset/sample` - Get sample dataset
- `POST /api/v1/dataset/upload` - Upload custom dataset
